# SabageManager

サバゲーサークルの当日チーム振り分けを自動化するWebアプリ（MVP）

## 技術スタック

- **フレームワーク**: SvelteKit + TypeScript
- **スタイリング**: Tailwind CSS
- **DB（予定）**: Firebase Firestore

## セットアップ

```bash
# 依存パッケージのインストール
npm install

# 開発サーバー起動（http://localhost:5173）
npm run dev

# ビルド
npm run build
```

## 機能（MVP）

### 1. チェックイン画面（`/`）
- メンバー名簿の一覧表示（名前・累計参加回数・経験レベル）
- 当日の参加登録（自前装備 / レンタルを選択）
- 参加取り消し

### 2. ダッシュボード（`/dashboard`）
- 参加人数・自前/レンタル内訳のサマリー
- チーム分け実行ボタン
- 赤チーム / 黄チームの振り分け結果表示

## チーム振り分けアルゴリズム

各メンバーのスコアを計算し、両チームの合計スコアが均等になるよう貪欲法で割り当てます。

| 条件 | スコア |
|------|--------|
| 自前装備 | +2点 |
| レンタル装備 | +1点 |
| 累計参加5回以上 | +1点 |
| 累計参加10回以上 | +2点 |

同スコアグループ内をランダムシャッフルすることで、毎回異なる組み合わせが生成されます。

## Firebase連携（次フェーズ）

`src/lib/firebase.ts` に設定の雛形があります。
Firebase Console でプロジェクトを作成し、設定値を入力してください。
